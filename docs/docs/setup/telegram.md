# Setup for Telegram Sync

## Steps

1. Head to [https://my.telegram.org/apps](https://my.telegram.org/apps) and create an app.

2. Copy the **API ID** and **API Hash** and save it for later to use in "Add source" process during the source setup.

3. In the Telegram App, navigate to [**Botfather**](https://t.me/BotFather) and create a new bot.

4. Copy the **Bot Token** and save it for later to use in "Add source" process during the source setup.
